Consumo real por SKU

Selecciona una orden abierta para capturar el consumo real.

Cómo usarlo

  • Selecciona una orden en estado CREADA o EN_PRODUCCIÓN.
  • Revisa el consumo teórico y ajusta cada SKU con el material real usado.
  • Usa Registrar consumo real para dejar la orden en producción.
  • Usa Registrar y cerrar orden cuando el lote ya esté terminado y validado.
  • El panel recalcula alertas, APU y margen con el consumo reportado por el operario.

Consejo operativo: si una línea queda sin editar, el sistema la toma con el valor teórico para no bloquear el proceso.
